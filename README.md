# monex
Кроссплатформенное мобильное приложение, разработанное на Flutter с использованием языка Dart, предназначенное для отображения актуальных курсов валют. Над проектом работают два разработчика — Баталин Роман и Илья Ключников. Приложение предоставляет удобный и интуитивно понятный интерфейс, поддерживает обновление курсов валют и ориентировано на учебное и практическое использование.
