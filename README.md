# monex
Проект представляет собой кроссплатформенное мобильное приложение на Flutter для просмотра курсов валют. Приложение показывает актуальные данные, имеет простой и удобный интерфейс. Пользователь может обновлять курсы и быстро находить нужную валюту. Проект разработан в учебных целях и помогает получить практические навыки мобильной разработки. Над проектом работают Баталин Роман и Илья Ключников, студенты группы 9-3-ПО-22.
