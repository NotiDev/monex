# monex
Кроссплатформенное мобильное приложение, разработанное на Flutter с использованием языка Dart, предназначенное для отображения актуальных курсов валют. Над проектом работают Баталин Роман и Илья Ключников, студенты группы 9-3-ПО-22. Приложение предоставляет удобный и интуитивно понятный интерфейс, поддерживает обновление курсов валют и подходит для учебного и практического использования.
